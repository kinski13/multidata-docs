# Data Sources

Below are the names of Feeds available for use when configuring data for export to the chain.

### DEX Connectors

- UniswapV2 - for quoting assets in UniswapV2-like swaps, such as UniV2, Sushi, Shiba, and Pancake
- UniswapV2LP - for quoting liquidity pool tokens at UniswapV2-like swaps, such as UniV2, Sushi, Shiba, and Pancake
- UniswapV3 - for quoting assets at UniswapV3 swap
- Bearing - for quoting bearing tokens
- CurveLP - for quoting liquidity pool tokens in curve.finance pools
- Curve - for quoting assets at curve.finance pools
- Yv - for quoting yv tokens
- Compound - for quoting Compound tokens

### Global Market

- TwelveData - for quoting stocks, fiat currencies, and commodities

### Other Feeds

- Mul - multiplying the price of one feed by another
- Median - finding the median value from multiple feed prices, with the option to set the count of allowable absent values
- Invert - 1 / feed price
- Alias - adding synonyms to feed names
- Constant - for integer constants
- Fallback - queries a list of feeds and returns the first successful response
