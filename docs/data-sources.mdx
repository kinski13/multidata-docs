# Data Sources

Below are the names of Feeds available for use when configuring data for export to the chain.

### DEX Connectors

- UniswapV2 - Quoting assets in UniswapV2-like swaps, such as UniV2, Sushi, Shiba, and Pancake.
- UniswapV2LP - Quoting liquidity pool tokens at UniswapV2-like swaps, such as UniV2, Sushi, Shiba, and Pancake.
- UniswapV3 - Quoting assets at UniswapV3 swap.
- Bearing - Quoting bearing tokens.
- CurveLP - Quoting liquidity pool tokens in curve.finance pools.
- Curve - Quoting assets at curve.finance pools.
- Yv - Quoting yv tokens.
- Compound - Quoting Compound tokens.

- TwelveData - Quoting stocks, fiat currencies, and commodities.

### Other Feeds

- Mul - Multiplying the price of one feed by another.
- Median - Finding the median value from multiple feed prices, with the option to set the count of allowable absent values.
- Invert - 1 / feed price
- Alias - Adding synonyms to feed names.
- Constant - Using integer constants.
- Fallback - Queries a list of feeds and returns the first successful response.
